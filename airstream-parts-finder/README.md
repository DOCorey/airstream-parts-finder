# 2020 Airstream Globetrotter Parts Finder

A searchable parts database built from the official 2020 Globetrotter Parts Book.

## Features
- 1,371 parts searchable by part number, description, and subsystem
- Filter by trailer model (23FB, 25FB, 27FB) and bed configuration (Queen/Twin)
- Diagram viewer — see where parts appear on the trailer
- One-click Google search with full vehicle context
- Copy part number to clipboard

## Local Development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and click **Add New Project**
3. Import your GitHub repository
4. Vercel will auto-detect Vite — click **Deploy**

That's it. Your app will be live at `your-project.vercel.app`.

## Updating

Any time you push changes to GitHub, Vercel automatically redeploys.
